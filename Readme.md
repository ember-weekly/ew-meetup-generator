ew-meetup-generator
=============

# Description

A commander CLI app

# Usage

To install ew-meetup-generator from npm, run:

```
$ npm install -g ew-meetup-generator
```

```node ./bin/ew-meetup-generator --help```

e.g. Meetups for the last 14 days:

```node ./bin/ew-meetup-generator -d 14````
